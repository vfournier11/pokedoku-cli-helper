# pokedoku CLI helper

Simple CLI helper for [pokedoku](https://pokedoku.com)

## Usage

Use the CLI helper to filter the list of Pokemon by ability, or type.

```bash
  go run main.go find-by-ability overgrow
  go run main.go find-by-type grass
```
